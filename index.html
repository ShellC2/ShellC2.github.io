中医代表落后和迷信吗

太上，不知有之。——《道德经》

善战者无赫赫之功。

上医治未病。

以上三句话，特别是后两句，其实内核是一模一样的，智者见于未萌，厉害的人总是把可能发展成危险的事物扼杀在萌芽状态，这样一来一般人反而看不出他的厉害。

如果一样东西，它被证明是有效的，长期如此。那么这样东西能被说是迷信吗？

而我们知道，认识是实践的产物，你看到了一个苹果并品尝了它，你就初步认识了它，别人告诉你说这是苹果，哦，你说，原来这就是苹果啊。名可名，非常名，当我们大家都不叫它苹果叫它红皮果，那么它就叫红皮果，可实际上，不管它叫什么名字，我们都知道它是它。别人把它放在你的面前，你会说，哦，这东西我看过，味道是甜的。这就是认识来源于实践。

为什么我们可以做到

发表于 2023-04-27 分类于 审问明辨
本文字数： 613 阅读时长 ≈ 1 分钟

换一个赛道，玩法不同了，局面便也大不同了。在智能手机出现之前，中国国产的手机行业存在感几乎为 0，进入智能手机的时代之后，虽然苹果呈一骑绝尘之势（中间被华为挑战了一下），但除了它，国外的手机品牌在中国一个能打的都没有，三星早已失色。在传统燃油车的时代，中国国产的汽车行业也好不到哪去，所谓市场换技术，这么多年下来，市场没了，技术却并没有跟上步伐，但是，新能源这块，我们绝对是领头羊的角色。2022年，中国汽车出口量超过德国成为世界第二大汽车出口国，今年第一季度，汽车出口量全球第一，照这个趋势下去，即使今年成不了全球第一，明年也该是了。日本的汽车产业也将和韩国的三星手机一样，虽然它仍然不会倒，但也就只能在中国以外的市场发威了，在中国，它们将拥有一个共同的名字，昨日黄花。

一切事物的发展都没有什么突然性，汽车行业的这场革命，布局从很早就开始了，现在只不过是到了出成果的时候。很多事便是这样，做着做着，便水到渠成。

那么，是不是换一个赛道，就对所有人都是公平的呢，明显不是，不管是智能手机，新能源汽车，还是移动互联网，全世界真正的玩家只有 2 个，其他人都掉队了。而这 2 个玩家，也至少有一个共同的特点，超大的内部市场。市场就在那里，就看你有无本领占有它。欧盟也有一个不亚于中、美的内部市场，但是这几次赛道变换，它都与新赛道无缘，所以，市场固然很重要，但它肯定不是唯一的决定性因素，那么问题来了——

为什么我们可以做到，为什么中国可以做到？

via browser1我是你的自来水。

为什么说它简洁？

  1. 安装包只有不到 2M 大小，与那些动辄 100M 左右大小的 浏览器 APP 相比，它显得如此另类，以至于你会怀疑它真的能用吗？
  2. 界面干净整洁，只有一个搜索框和一个 logo ,没有广告，没有新闻，没有其它，如果你想，甚至搜索框和 logo 都可以隐藏起来，一种极致纯粹的简单。

为什么说它强大？

  1. 自带广告过滤器 AdFilters，屏蔽规则多达5万条以上。

  2. 支持手动切换多种搜索引擎，google ，baidu ，bing ，sogou ，头条，神马，duckduckgo ，应有尽有。

  3. 高度定制化。logo ，搜索框，背景，自定义 CSS ，这些都可以私人定制，你可以充分魔改。
