via，一款简洁而强大的浏览器

发表于 2023-04-06 更新于 2023-05-23
本文字数： 910 阅读时长 ≈ 2 分钟

via browser1我是你的自来水。

为什么说它简洁？

  1. 安装包只有不到 2M 大小，与那些动辄 100M 左右大小的 浏览器 APP 相比，它显得如此另类，以至于你会怀疑它真的能用吗？
  2. 界面干净整洁，只有一个搜索框和一个 logo ,没有广告，没有新闻，没有其它，如果你想，甚至搜索框和 logo 都可以隐藏起来，一种极致纯粹的简单。

为什么说它强大？

  1. 自带广告过滤器 AdFilters，屏蔽规则多达5万条以上。

  2. 支持手动切换多种搜索引擎，google ，baidu ，bing ，sogou ，头条，神马，duckduckgo ，应有尽有。

  3. 高度定制化。logo ，搜索框，背景，自定义 CSS ，这些都可以私人定制，你可以充分魔改。

    • logo 可以隐藏，可以把它设定成你想要的图片，可以是文字，也可以是一段 HTML 代码，我自己就把它置换成了 HTML 代码：

      链接指向的是本博客地址，这样以后我想进入自己博客的时候只要点击一下我的浏览器自定义 logo 就行了，so easy. 当然，链接地址由你做主，你想指向哪就指向哪。

    • 搜索框可以是矩形，也可以是一条横线，你也可以设置透明度，这将在你更换背景时派上用场，比如你用一张图片做背景，如果你不想让图片的某一部分被搜索框遮挡，你就可以把它的透明度提高。搜索框的描边可以变换粗细，也同样支持设置透明度，搜索框设定成矩形时，如果你觉得矩形有点单调，可以将矩形导圆角。

    • 背景可以切换成不同的纯色，当然，也支持用任意一张图做背景。

    • 除了这些，你还可以自己添加脚本，添加广告屏蔽规则，浏览器本身还有资源嗅探功能。总之，别的浏览器有的，它也有，别的浏览器不给你的，它给你。

你安全吗？

除了简洁和强大，它还非常注重用户的隐私。它所需要的权限非常非常少。

太上，不知有之。

是的，你在使用它，但它却想让你忽略它，事了拂衣去，深藏功与名。
